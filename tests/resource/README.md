# 测试资源

此目录包含CodeAgent测试所需的资源文件。

## 文件列表

### test_students.xlsx
- **类型**: Excel文件
- **用途**: Excel数据统计测试（`test_excel_statistics`）
- **内容**: 8名学生成绩数据
- **列**: 姓名、年龄、语文、数学、英语、总分、班级
- **创建**: `tests/create_test_excel.py`

### test_text.txt
- **类型**: 文本文件
- **用途**: 文本分析测试（`test_text_analysis`）
- **内容**: Python编程介绍（10行）
- **统计**: 594字符（含空格），495字符（不含空格），99单词，16数字

## 重新生成测试资源

如果需要重新生成测试文件，运行以下脚本：

```bash
# 生成Excel文件
uv run python tests/create_test_excel.py

# 计算文本统计
uv run python tests/calculate_text_stats.py
```
